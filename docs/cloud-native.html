技术栈 · Kubernetes

概述

Kubernetes 已成为智算集群的标准平台。本文介绍 Kubernetes 深度定制、Operator 开发、CRD 和自动化运维的实践经验。

Operator 开发

Operator 是 Kubernetes 的扩展机制，用于封装运维知识：

  • 使用 Operator SDK 快速开发
  • 通过 CRD 定义自定义资源
  • Reconcile 循环实现期望状态

Helm 包管理

Helm 是 Kubernetes 的包管理器：

# 添加 Chart 仓库
helm repo add bitnami https://charts.bitnami.com/bitnami

# 安装 Chart
helm install my-release bitnami/nginx

# 创建自定义 Chart
helm create my-chart

Istio 服务网格

Istio 提供微服务治理能力：

  • mTLS 双向认证
  • 流量管理（路由、熔断）
  • 可观测性（遥测、追踪）
