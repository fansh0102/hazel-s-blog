🔌 高性能网络

技术栈 · RDMA

概述

高性能网络是智算集群的关键基础设施。本文介绍 InfiniBand/RoCE 配置和 GPUDirect RDMA 技术。

InfiniBand vs RoCE

  • InfiniBand (IB)：专用高速网络，延迟低但成本高
  • RoCE (RDMA over Converged Ethernet)：以太网承载 RDMA，兼容现有网络

GPUDirect RDMA

GPUDirect RDMA 允许 GPU 直接跨节点通信，绕过 CPU：

  • 减少数据传输延迟
  • 降低 CPU 利用率
  • 提升分布式训练效率 30%+

MLNX_OFED 配置

./mlnx_ofed_install.py --upstream-libs
