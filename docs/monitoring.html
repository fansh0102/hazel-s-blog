📊 监控运维

技术栈 · Prometheus

概述

智算集群监控是保障稳定运行的关键。本文介绍基于 Prometheus + Grafana + DCGM 的监控体系建设。

DCGM 集成

DCGM (Data Center GPU Manager) 是 NVIDIA 的 GPU 监控工具：

  • GPU 利用率、显存、温度、功耗
  • NVLink 带宽监控
  • ECC 错误统计

Prometheus 指标采集

配置 Prometheus 采集 GPU 指标：

scrape_configs:
  - job_name: 'gpu-metrics'
    static_configs:
      - targets: ['localhost:9400']

Grafana 可视化

推荐 GPU 监控面板：

  • GPU Utiliztion - 利用率
  • GPU Memory Used - 显存
  • GPU Temperature - 温度
  • GPU Power Draw - 功耗

告警规则

关键告警配置：

  • GPU 利用率持续 < 20%（资源浪费）
  • 显存使用 > 90%（OOM 风险）
  • GPU 温度 > 85°C（过热）
  • GPU 不可用（节点故障）
